Night Mode Theme

The app enables a user friendly basic night mode theme for Odoo 14.0 community edition. Enabling the Night mode theme in Odoo is very simple. To enable, Click on the night mode icon displayed in the header section.To disable, Click again the same icon.

Features

  • Easy to enable / disable night mode.
  • Night mode for list,kanban,graph,pivot and form view.
  • Night mode according to user's preference (User's preference).

  • Available for all basic modules excluding website and POS.
  • Improved readability of text.
  • Better Contrast and Less Blue light.
